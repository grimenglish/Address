# 식혜명가 주소록 즐겨찾기

Supabase 영구저장 주소록 + CJ택배 엑셀 다운로드 앱입니다.

## 파일 구성
- `app.py`
- `requirements.txt`
- `README.md`

## 기능
- 지인 / 거래처 폴더 분리
- 지인은 파란색, 거래처는 주황색으로 표시
- 전체 / 지인 / 거래처 탭으로 주소록 분리 조회
- 이름 / 전화번호 / 우편번호 / 주소 / 수량 / 상품명 / 배송메세지 / 메모 저장
- Supabase DB 영구저장
- 발송 엑셀 다운로드 화면에서 왼쪽 주소록 → 오른쪽 발송목록으로 추가
- 엑셀 출력 전 상품명 / 수량 / 배송메세지 수정
- 수정내용을 Supabase 기본값으로 저장 가능
- CJ택배 업로드용 `Delivery` 시트 엑셀 다운로드
- CSV 백업 / 가져오기

## Supabase SQL
```sql
create table if not exists addresses (
  id bigint generated by default as identity primary key,
  category text default '지인',
  name text not null,
  phone text,
  zipcode text,
  address text not null,
  quantity integer default 1,
  product_name text,
  delivery_message text default '문 앞',
  memo text,
  created_at timestamp with time zone default now()
);

alter table addresses enable row level security;
```
