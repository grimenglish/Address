# 식혜명가 주소록 즐겨찾기

Supabase에 주소록을 영구 저장하고, 지인/거래처를 폴더처럼 관리한 뒤 필요한 사람만 골라 CJ택배 업로드용 엑셀로 내려받는 Streamlit 앱입니다.

## 파일 구성
- `app.py`
- `requirements.txt`
- `README.md`

## 주요 기능
- 지인 / 거래처 폴더 관리
- 이름 / 전화번호 / 우편번호 / 주소 / 수량 / 상품명 / 배송메세지 / 메모 저장
- Supabase DB 영구 저장
- 검색 / 수정 / 삭제
- 엑셀 출력 전 상품명 / 수량 / 배송메세지 임시 수정
- 수정한 내용을 주소록 기본값으로 저장 가능
- 선택한 주소만 CJ택배 양식 엑셀 다운로드
- CSV 백업 / 가져오기

## Streamlit Secrets
Streamlit Cloud → Manage app → Settings → Secrets 에 아래처럼 입력하세요.

```toml
SUPABASE_URL = "https://xxxxx.supabase.co"
SUPABASE_KEY = "sb_secret_xxxxx"
APP_PASSWORD = "원하는비밀번호"
```

## Supabase SQL
SQL Editor에서 아래 테이블이 있어야 합니다.

```sql
create table if not exists addresses (
  id bigint generated by default as identity primary key,
  category text default '지인',
  name text not null,
  phone text,
  zipcode text,
  address text not null,
  quantity integer default 1,
  product_name text,
  delivery_message text default '문 앞',
  memo text,
  created_at timestamp with time zone default now()
);

alter table addresses enable row level security;
```

## 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```
